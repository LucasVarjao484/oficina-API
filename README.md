# API de oficina

Este projeto busca criar uma API RESTful de uma oficina para conserto, manutenção e serviços gerais para veículos

# Tecnologias utilizadas

- Java 21
- Spring Boot e Spring Framework
- PostgreSQL
- Flyway
- OpenAPI & Swagger


# Como executar a API

- Certifique-se que as versões e dependências do projeto estão corretas com o Maven
- Utilize as variáveis de ambiente no application.yaml para se conectar com o banco de dados
- Utilize um client como Postman, Insomnia, etc, para fazer as requisições
